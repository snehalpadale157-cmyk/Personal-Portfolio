# Personal Portfolio

## Overview

This is my personal portfolio website built using HTML5 and CSS3. It showcases my profile, projects, and contact information using a clean, mobile-first responsive design.

## Features

* Hero section with name and tagline
* About Me section with profile photo and short bio
* Projects section with three project cards
* HTML-only contact form
* Mobile-first responsive design

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries
